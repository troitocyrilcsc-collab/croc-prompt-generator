# Générateur de Prompt CROC

Outil simple pour créer des prompts IA avec la méthode **CROC**.

## Méthode

C → Cadrage  
R → Rôle  
O → Objectif  
C → Cible  

## Utilisation

1. Remplir les champs
2. Cliquer sur "Générer"
3. Copier le prompt

## Hébergement

Compatible avec **GitHub Pages**.
